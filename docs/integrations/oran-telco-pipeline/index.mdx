## Pipeline

| Stage     | Reference behavior                                                         |
| --------- | -------------------------------------------------------------------------- |
| Export    | A separate adapter or exporter emits files, HTTP payloads, or metrics.     |
| Normalize | Parse authored fields and attach site metadata.                            |
| Label     | Apply example thresholds as neutral review fields.                         |
| Route     | Shape separate payloads for observability, storage, and analytics outputs. |

Expanso does not provide an O-RAN protocol collector in this example. OpenShift, OTLP, Prometheus, object storage, and Cloudera are external deployment and service boundaries.

## Limitations and assumptions

<Limitations>

- All telemetry, thresholds, topology, and destination contracts are illustrative.
- Timing, radio quality, and congestion labels are not validated engineering conclusions.
- The repository has not exercised a RAN source, OpenShift deployment, destination, outage, or operating envelope.
- Delivery consistency depends on each output's acknowledgement, retry, and buffering configuration.

</Limitations>
