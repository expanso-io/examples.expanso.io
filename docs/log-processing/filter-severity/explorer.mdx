Choose a stage, compare the retained records, and inspect the severity predicate. Evaluate the omitted records as carefully as the output.

## Limitations

The displayed selection is not an executed result. Source mappings, malformed records, retention obligations, delivery failure, and volume effects are not tested. Review the [complete configuration](/log-processing/filter-severity/complete-filter-severity) before adapting it.
