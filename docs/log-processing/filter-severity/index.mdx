## Pipeline

| Stage     | Intended change                                             |
| --------- | ----------------------------------------------------------- |
| Parse     | Attempt JSON parsing and preserve a fallback message.       |
| Normalize | Map the source level to the configured severity vocabulary. |
| Select    | Keep records that satisfy the explicit severity predicate.  |
| Route     | Send retained records to the configured local outputs.      |

Filtering changes observability as well as volume. A severity label alone does not establish whether an event can be discarded safely.

## Limitations and assumptions

- The shown selection is curated and has no maintained execution evidence.
- Source-specific level mappings, malformed lines, multiline logs, retention, and failure delivery are not assessed.
- No storage, bandwidth, query, or cost reduction is measured.
