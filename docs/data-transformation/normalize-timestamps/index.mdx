## Pipeline

| Stage     | Intended change                                              |
| --------- | ------------------------------------------------------------ |
| Parse     | Select a parser for the declared source representation.      |
| Normalize | Convert the parsed instant to a UTC representation.          |
| Enrich    | Derive configured calendar fields from the normalized value. |

Timezone offsets identify an instant; timezone names can also carry daylight-saving rules. Treat values without either as ambiguous unless the source contract supplies a zone.

## Limitations and assumptions

- The displayed outputs are curated and have no maintained execution evidence.
- Invalid dates, leap seconds, ambiguous local times, timezone-database versions, and clock quality are not assessed.
- Derived calendar fields reflect the configured timezone and should not silently replace source context.
