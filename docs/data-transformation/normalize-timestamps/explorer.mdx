# Interactive Timestamp Normalization Explorer

**See timestamp normalization in action!** Use the interactive explorer below to step through 3 stages of converting mixed timestamp formats into standardized ISO 8601 UTC. Watch how consistent timestamps enable time-based queries and aggregations.

## How to Use This Explorer

1. **Navigate** using arrow keys (← →) or click the numbered stage buttons
2. **Compare** the Input (left) and Output (right) showing normalization at each stage
3. **Observe** how different formats get unified
4. **Inspect** the YAML code showing the parsing and conversion logic
5. **Learn** from the stage description explaining the benefits
