## Pipeline

| Format   | Contract to review                                                |
| -------- | ----------------------------------------------------------------- |
| Avro     | Writer schema, reader compatibility, and registry ownership.      |
| Parquet  | Row grouping, column types, partitioning, and consumer support.   |
| Protobuf | Field numbers, optionality, generated types, and evolution rules. |
| JSON     | Naming, numeric precision, null behavior, and validation.         |

Binary output that appears unreadable in a console is not proof of a correct conversion. Verify it with the intended decoder and schema.

## Limitations and assumptions

- No maintained machine gate encodes and decodes these configurations end to end.
- Compression, size, throughput, query behavior, and compatibility are not measured.
- Schemas and outputs are representative fixtures; consumers and registries are not included.
