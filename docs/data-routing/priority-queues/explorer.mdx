# Priority Queue Explorer

Use the stage controls to compare the synthetic input, authored output, and configuration at each step.

Select a stage, inspect the authored highlights, then open the configuration that produces the illustrated shape.

## Limitations

The stages label and route records; they do not execute a scheduler or demonstrate queue fairness, capacity, or service levels.
