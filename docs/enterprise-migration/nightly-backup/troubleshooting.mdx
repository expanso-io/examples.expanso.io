# Troubleshoot the Nightly Backup Reference

| Symptom                     | Check                                                                                      |
| --------------------------- | ------------------------------------------------------------------------------------------ |
| Database connection fails   | Network path, TLS, credentials, and table permissions.                                     |
| Incremental result is empty | Watermark column, timezone, late updates, and source transaction boundary.                 |
| Checksum changes            | Field selection, serialization order, nulls, numeric representation, and digest algorithm. |
| Object write fails          | Bucket, region, credentials, encryption policy, object path, and retry behavior.           |
| Restore count differs       | Snapshot completeness, duplicate keys, object inventory, rejects, and restore ordering.    |

Diagnose with synthetic data and a disposable destination. Restore to a staging table and compare counts and digests before planning any destructive swap.

**Reference:** [Open the combined configuration](/enterprise-migration/nightly-backup/complete-pipeline).
