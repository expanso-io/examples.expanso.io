| --------------- | ----------------------------------------------------- |

| Customer fields | Apply the same pattern to identity fields.            |
| Location fields | Separate retained geography from protected values.    |

The Explorer shows authored transformations; it does not test cryptographic key operations or a deployed threat model.

## Limitations and assumptions

<Limitations>

- The records and keys shown in the example are synthetic.
- No current machine gate executes encryption, decryption, key rotation, or recovery.
- Algorithm choice, nonce handling, key scope, access, logging, and failure behavior require deployment-specific review.
- This example does not establish a security or regulatory posture.

</Limitations>
