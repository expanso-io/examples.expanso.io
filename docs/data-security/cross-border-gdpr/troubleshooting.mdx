# Cross-Border Minimization Troubleshooting

## Expected fields remain

**Check:** Compare the input schema with every deletion and assignment path. Nested fields require their exact path. Add a synthetic rejected record before changing a real source.

## Derived identifiers change between runs

**Check:** Confirm that each process receives the intended environment-managed salt. A missing or different salt changes the digest. Do not print the salt or use the demonstration fallback in a deployment.

## Records use the wrong destination

**Check:** Inspect the authored region field and branch order. A field label does not prove physical location. Verify the actual service endpoint and surrounding transfer context outside this example.

## Output still appears identifiable

**Check:** Treat pseudonymized output as potentially personal data. Review retained fields, linkability, source access, other datasets, purpose, retention, and deletion with privacy and legal reviewers.
