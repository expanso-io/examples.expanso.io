# Cross-Border Minimization Review

The example removes, generalizes, and pseudonymizes selected fields before an external output. It does not determine whether the resulting record is identifiable or suitable for a transfer.

<ExampleSurface
  kind="reference"
  title="Before you adapt it"
  description="Resolve the example boundary before changing processors or destinations."
>

- An Expanso environment for configuration inspection
- A field inventory and documented purpose for retained data
- Privacy and legal review of the complete source-to-destination flow

</ExampleSurface>

## Adaptation checks

- Replace demonstration salts with environment-managed secrets.
- Assess linkability across retained and external datasets.
- Review purpose, access, retention, deletion, destination, and transfer context.

This route is a configuration review, not a verified execution guide.
