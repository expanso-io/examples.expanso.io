# Interactive PII Removal Explorer

**See PII removal in action!** Use the interactive explorer below to step through 6 stages of data transformation. Watch as credit cards, emails, IP addresses, and other PII are progressively removed or anonymized.

## How to Use This Explorer

1. **Navigate** using arrow keys (← →) or click the numbered stage buttons
2. **Compare** the Input (left) and Output (right) JSON at each stage
3. **Observe** how fields are removed (red strikethrough) or transformed (green highlight)
4. **Inspect** the YAML code showing exactly what processor was added
5. **Learn** from the stage description explaining the technique and compliance benefit
