# Interactive Payment PII Deletion Explorer

**See PCI-DSS compliance in action!** Use the interactive explorer below to step through 4 stages of payment data sanitization. Watch how sensitive credit card information is systematically removed while preserving analytics capabilities.

## How to Use This Explorer

1. **Navigate** using arrow keys (← →) or click the numbered stage buttons
2. **Compare** the Input (left) and Output (right) showing PII removal at each stage
3. **Observe** how sensitive fields are deleted (highlighted in red/green)
4. **Inspect** the YAML code showing exactly what data was removed
5. **Learn** from the stage description explaining the compliance benefit
